# v0 Archive

This directory is a curated snapshot of the relevant material from
`/home/tack/code/villmager_v0/`, flattened slightly for reference inside the
new `villmage` repo.

It is intended as historical context, not as the active project layout.

## Layout

- `docs/` preserves the original planning and implementation notes.
- `src/` keeps the flat Python prototype runtime together in one place.
- `cost_analysis/` preserves the old auxiliary analysis package and its breakdowns.
- `utils/` preserves one-off developer utilities from the old repo.
- `transcripts/` preserves generated transcripts from earlier runs.

See [DIRECTORY_MAP.md](/home/tack/code/villmage/v0/DIRECTORY_MAP.md) for the
old-to-new path mapping and the list of intentionally omitted directories.
